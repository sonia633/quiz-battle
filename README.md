# AI Quiz Battle

<!-- run-site -->
## ▶️ Voir le site (en local)

```bash
cd ai-service
pip install -r requirements.txt
uvicorn app.main:app --port 8000
```

Puis ouvrir dans le navigateur : **http://localhost:8000/docs**

> Service IA Python (les services Java + l'app Flutter nécessitent JDK/Flutter).

Plateforme de quiz multijoueur (type Kahoot) — monorepo multi-stack (Flutter · Spring Boot · FastAPI · PostgreSQL).
